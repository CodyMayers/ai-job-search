# ai-job-search
Scrapes job listing data from Indeed for a given search term and location (or "remote")


Find the path to your default Google Chrome profile
C:\Users\YOUR_USER_NAME\AppData\Local\Google\Chrome\User Data\Default

Find the path to your Google Chrome executable
Most likely "C:\Program Files\Google\Chrome\Application\chrome.exe"

In a cmd terminal, open chrome in debug mode
"C:\Program Files\Google\Chrome\Application\chrome.exe" --remote-debugging-port=9222 --user-data-dir="C:\Users\mayer\AppData\Local\Google\Chrome\User Data\Default"

In the Chrome window that opens, sign into Chrome and then Indeed
Leave the window open, then run the scraper
